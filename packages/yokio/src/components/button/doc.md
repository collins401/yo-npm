# 按钮button

## 基本属性

|  属性  | 说明 | 类型  |     默认值      |
| :------: | :------: | :-----: | :-------------: |
|   type   |  按钮类型，可选值为 <br />  `primary`、`danger`、`warn`  | string | default |
|  size    | 按钮大小，可选值为<br />`large`、`small`  |  string  |       |
| plain  | 按钮镂空  |  false  |   false    |
| disabled | 按钮禁止点击效果  |  false  |   false    |
|  block   | 按钮长度100%  |  false  |   false    |
